Hello!

Lokra UI Starter is admin template ready for mobile & desktop application, based on the Angular2 framework.

This document will help you setup this starter kit and find a guide about the functionality, components, structure, workflow.

Setup

Prerequisites:

  • Node.js
  • NPM
  • angular-cli
Command
# Development server
npm start

# Build developer package
npm run "build:dev-aot"

# Build production package
npm run "build:prod-aot"

# Build developer package without AOT
npm run "build:dev"

# Build production package without AOT
npm run "build:prod"

# Get some help
ng help

Further help

To get more help on the `angular-cli` use `ng --help` or go check out Angular CLI

Layout Engine

For the layout, the template uses @angular/flex-layout angular 2 component layout engine which uses flexbox css. For detailed usage information you can read the docs over at flex-layout

Theming

For detailed custom theming guide you can read the offical material 2 docs

Structure

Template file structure is base on angular-cli generate project.

- ui-starter                          # project root folder.
  |- docs                             # documents folder.
  |- src                              # source code folder.
     |- app                           # application root.
        |- pages ...                  # template page component folder ...
        |- shared                     # common function or common component with 3rd party
         - app.component.html         # main container template
         - app.component.scss         # main container style sheet
         - app.component.spec.ts      # main unit test
         - app.component.ts           # bootstrap component
         - app.module.ts              # bootstrap module
         - app.routes.ts              # global routes
         - app.service.ts             # global state service
         - app.theme.bootstrap.scss   # bootstrap 4 style sheet
         - app.theme.material.scss    # material 2 style sheet
         - app.theme.custom.scss      # customize style sheet
         - app.theme.scss             # style sheet mixin
     |- assets                        # static resources
     |- electron                      # electron support file
     |- environments                  # build environment configurations
      - favicon.ico                   # icon
      - index.html                    # bootstrap index page, manage by webpack
      - main.ts                       # angular import configuration
      - polyfills.ts                  # angular dependencies import configuration
      - styles.scss                   # global style sheet configuration
      - test.ts                       # unit test configuration
      - tsconfig.json                 # typescript compile configuration
      - typings.d.ts                  # typescript definitions configuration
   - .editorconfig                    # IDE edit support
   - .gitignore                       # git ignore file
   - angular-cli.json                 # angular-cli global configuration
   - config.xml                       # cordova configuration
   - karma.conf.js                    # unit test global configuration
   - package.json                     # npm dependencies manage
   - protractor.conf.js               # e2e test global configuration
   - tslint.json                      # typescript code lint configuration

CSS Framework

This starter kit is composed of several different style frameworks:

  • Bootstrap 3
  • Bootstrap Material Design
  • Angular Material Library

We have modified the style frame for the Material style, and you can mix and match the style properties of the different frames.

Support
